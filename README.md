# Swift-baseball

# 게임 세팅
## Layout
- [ ] +와 - 버튼을 구현한다.
- [ ] 게임 시작 버튼을 구현한다.
- [ ] 목숨의 갯수가 보이는 View를 구현한다.
- [ ] 게임 사진을 View를 구현한다.
- [ ] "숫자 야구 게임"을 View를 구현한다.
- [ ] 게임 화면에서 남은 목숨 View를 구현한다.
- [ ] 숫자 입력 칸과 도전 버튼을 구현한다.
- [ ] Toast를 구현한다.
- [ ] Dialog를 구현한다. 

## Controller 
- [ ] 게임 시작 버튼을 누르면 화면이 넘어간다.
- [ ] + - 버튼을 누르면 목숨 값이 수정이 된다.

<br><br><br>

## LifeCount
- [ ] 목숨은 1로 시작한다.
- [ ] 목숨은 1개 이상 10개 이하이다.
- [ ] 목숨이 0개이면 알려준다.

<br><br><br>

# 예외처리
## LifeCount
- [ ] 목숨이 0개 이하면 안된다.
- [ ] 목숨이 10개 이상이면 안된다.
